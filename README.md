# 单词仓库 - 儿童英语词汇学习工具

为4-9岁儿童和家长打造的英语词汇学习工具，通过AI生成的可爱闪卡让学习更有趣。

## 功能特点

- 🎨 AI智能图片生成
- 📚 预制词库支持
- 💾 本地数据持久化
- 🔄 智能图片共享池
- ☁️ 云端同步准备
- 👥 多用户隔离

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- IndexedDB
- AI图片生成

## 开发

```bash
npm install
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看应用。
